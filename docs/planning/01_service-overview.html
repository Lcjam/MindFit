서비스 개요

심리상담 전문가 마켓플레이스 MindFit의 서비스 정의, 포지셔닝, 핵심 가치 및 MVP 범위 기획서

서비스 기획 포지셔닝 마켓플레이스 멘탈헬스 v0.1 Draft
Section 01

서비스 정의

한 줄 정의

MindFit은 검증된 심리상담 전문가와 도움이 필요한 내담자를 연결하는 심리상담 버티컬 마켓플레이스입니다.

배달의민족·쿠팡이츠가 음식 공급자(식당)와 수요자(소비자)를 연결하듯, MindFit은 공급자(임상·상담심리 전문 상담사)와 수요자(우울·번아웃 등을 겪는 직장인)를 디지털 플랫폼으로 연결합니다. 자체 채팅 시스템을 기반으로 텍스트 상담 또는 Zoom 링크 공유를 통한 화상 상담을 지원합니다.

🏗️

플랫폼 구조

양면 시장(Two-sided Marketplace). 상담사와 내담자 두 고객 집단 모두를 위한 가치를 창출하며, 플랫폼이 중간에서 신뢰·결제·커뮤니케이션 인프라를 제공합니다.

💬

상담 진행 방식

자체 채팅 시스템(실시간) 기반. 상담사가 Zoom 링크를 채팅으로 공유하거나 텍스트 상담으로 진행합니다. 외부 화상회의 API는 연동하지 않습니다.

📋

문진표 시스템

내담자가 상담 신청 시 문진표를 작성하며, 결제 완료 후 해당 상담사만 열람 가능합니다. 개인정보 보호와 신뢰 형성의 핵심 장치입니다.

🎓

상담사 자격 검증

임상·상담심리 석사 이상 학위 보유자만 등록 가능. 자격증·학위 서류를 업로드하면 관리자가 직접 승인합니다. 플랫폼 신뢰의 핵심 차별점입니다.

Section 02

해결하는 문제

공급 측 — 상담사

상담사가 겪는 문제

  • 개인 상담사의 마케팅·홍보 역량 부재 — 임상 전문가가 마케팅을 별도로 배워야 하는 구조
  • 클라이언트 예약·결제·일정 관리를 모두 개인이 수동으로 처리
  • 신뢰 없는 초기 내담자 확보의 어려움 — 온라인 후기·신뢰 시스템 부재
  • 자격을 갖춘 전문가와 무자격자가 온라인에서 혼재 — 전문성 희석
  • 소규모 개인 개업 시 초기 고정비(공간 임대 등) 부담
수요 측 — 내담자

내담자가 겪는 문제

  • 어떤 상담사가 실제로 자격을 갖췄는지 검증 불가 — 사기·부실 상담 위험
  • 상담 예약 프로세스의 높은 마찰 — 전화 문의, 이메일, 긴 대기 시간
  • 가격 정보 불투명 — 상담 전까지 비용을 알기 어려움
  • 우울·번아웃 상태에서 "상담받겠다"는 결심을 실행으로 연결하는 행동력 저하
  • 멘탈 케어에 대한 사회적 낙인(stigma) — 상담을 받는다는 사실이 알려질까 하는 두려움
Section 03

핵심 가치 3가지

01
접근성 (Accessibility)

웹·모바일 웹을 통해 언제 어디서나 상담사를 탐색하고 즉시 예약 가능. 복잡한 절차 없이 몇 번의 탭으로 상담 예약 완료. 행동력이 저하된 사용자도 진입할 수 있는 초저마찰 UX.

신뢰 (Trust & Verification)
02
신뢰 (Trust & Verification)

임상·상담심리 석사 이상 + 관리자 서류 승인 시스템으로 자격 미달 상담사 원천 차단. 투명한 가격 공개, 리뷰 시스템으로 선택의 근거 제공.

프라이버시 (Privacy)
03
프라이버시 (Privacy)

문진표는 결제 완료 후 해당 상담사만 열람 가능. 익명성 보장, 낙인 감소 설계. 상담 기록과 개인정보를 엄격히 보호하여 사용자가 심리적 안전감을 느끼도록 설계.

Section 04

경쟁 포지셔닝

구분 마인드카페 트로스트 MindFit
핵심 포지셔닝 콘텐츠·커뮤니티 중심 채팅 상담 중심 전문 자격 상담사 마켓플레이스
주요 서비스 정서 관리 콘텐츠, 커뮤니티, 상담 연결 구독형 텍스트 채팅 상담 상담사 탐색 → 예약 → 결제 → 채팅/화상 상담
상담사 자격 기준 다양한 수준 (낮음~중간) 심리사 자격 확인 (중간) 임상·상담심리 석사 이상 + 서류 심사 (높음)
수익 모델 B2C 구독 + 광고 B2C 구독 세션 중개 수수료 (20%)
가격 투명성 낮음 정액제 (중간) 상담사 자율 책정, 사전 공개
프라이버시 설계 보통 보통 문진표 접근 권한 분리, 익명성 강조
핵심 차별점 풍부한 콘텐츠 생태계 낮은 진입 장벽의 채팅 자격 검증 + 투명한 마켓플레이스 구조
Section 05

언어 전략 — "치료"가 아닌 "상담"

사용 금지
치료 / 진료 / 처방
→
사용 권장
상담 / 코칭 / 대화
⚖️

의료법 저촉 회피

"치료", "진료", "처방" 등의 표현은 의료법상 의사·정신건강의학과 전문의의 영역입니다. MindFit은 의료 서비스가 아닌 심리 상담 서비스임을 명확히 해야 규제 리스크를 피할 수 있습니다.

🏷️

낙인(Stigma) 감소

"치료"라는 단어는 "나는 병자다"라는 인식을 유발해 잠재 사용자의 진입 장벽을 높입니다. "상담", "코칭", "대화"는 일상적이고 긍정적인 자기 계발 프레임으로 접근성을 높입니다.

💡

카피라이팅 원칙

모든 UI 문구, 마케팅 텍스트, 이메일에서 의료적 언어를 배제합니다. "마음이 힘들 때, 전문가와 대화하세요" 방향으로 통일합니다.

Section 06

핵심 UX 인사이트

Core Insight
"우울·번아웃 사용자는 행동력이 저하되어 있다. UX 마찰을 극단적으로 줄이지 않으면 이탈한다."

우리의 핵심 타겟(우울·번아웃 직장인)은 정상적인 컨디션의 사용자가 아닙니다. 회원가입 절차가 길면 포기하고, 결제 단계가 복잡하면 이탈하며, 상담사 탐색이 어려우면 "나중에 하지 뭐"라고 미루게 됩니다. 그 "나중"은 오지 않을 수 있습니다.

⚡

3분 예약 원칙

첫 방문 후 3분 이내에 상담사를 선택하고 예약을 완료할 수 있어야 합니다. 불필요한 정보 입력, 복잡한 인증 절차는 모두 제거 대상입니다.

🎯

결정 피로 최소화

상담사 목록에 너무 많은 선택지를 주면 오히려 결정하지 못합니다. 개인화 추천·필터·태그를 활용해 적정 수의 상담사를 빠르게 매칭합니다.

🛡️

심리적 안전감 제공

문진표 내용이 상담사 외 타인에게 공개되지 않는다는 점을 UI에서 명시적으로 보여줘야 합니다. 불안 요소를 사전에 해소하는 것이 전환율에 직접 영향을 줍니다.

Section 07

MVP 범위 vs 향후 확장

MVP — 핵심 기능
  • 상담사 회원가입 + 자격 서류 업로드 + 관리자 승인
  • 내담자 회원가입 + 프로필 설정
  • 상담사 탐색 (키워드·태그·가격 필터)
  • 상담사 프로필 상세 페이지 (자격증, 소개, 가격, 리뷰)
  • 상담 신청 + 문진표 작성
  • 결제 처리 (PG 연동)
  • 자체 실시간 채팅 시스템
  • 상담 후 리뷰 작성
  • 상담사 정산 (수수료 20% 공제 후 정산)
  • 관리자 대시보드 (상담사 승인/관리, 정산 관리)
향후 확장 방향
  • AI 기반 상담사 매칭 추천 알고리즘
  • B2B EAP(Employee Assistance Program) 기업 계약
  • 정기 구독 플랜 (월 N회 상담 패키지)
  • 프리미엄 상담사 노출(Featured Listing) 광고 상품
  • 그룹 상담 / 워크숍 예약 기능
  • 심리 검사 도구 연동 (PHQ-9, GAD-7 등)
  • 상담 일지 / 진행 기록 관리
  • 네이티브 앱 (iOS / Android)
  • 다국어 지원 (영어, 일본어)
  • 보험사·복지 포인트 연동 결제
