내담자 (Client) 기능

우울증·번아웃 직장인을 1차 타겟으로 하는 서비스 이용자

1
회원가입 & 인증
기능 항목상세 설명우선순위
이메일 + 비밀번호 가입 표준 이메일 형식 검증, 비밀번호 강도 규칙 적용 Must
이메일 인증 가입 후 인증 메일 발송 → 링크 클릭 시 계정 활성화. V1 최우선으로 이동 — MVP 범위에서는 미구현 (12번 마스터 플랜 §11, 2026-07-07 재산정) V1
온보딩 정보 입력 이름, 생년월일 입력 (간단 스텝 UI) Must
소셜 로그인 카카오 OAuth 연동 (한국 1순위 — 로그인 UI에서 카카오 버튼을 최상단에 배치). 구글 OAuth는 v1.0 이후 추가. V1 최우선으로 이동 — MVP 범위에서는 미구현 (12번 마스터 플랜 §11) V1
2
상담사 탐색 & 검색
기능 항목상세 설명우선순위
상담사 목록 조회 카드 형태 그리드 UI, 사진·이름·전문 분야·가격·별점 표시 Must
필터링 전문 분야(우울 / 번아웃), 가격대(범위 슬라이더), 상담 방식(채팅 / 화상링크)
  • 필터 복합 적용 지원
  • 결과 없을 시 안내 메시지 표시
 Must
상담사 프로필 상세 자격 뱃지, 자기소개, 상담 방식, 가격, 가능 시간, 리뷰 목록
  • 자격 뱃지: 관리자 승인 완료 상담사만 표시
  • 리뷰: 익명 처리 옵션 반영
 Must
3
예약 & 결제 플로우
기능 항목상세 설명우선순위
날짜 / 시간 선택 상담사 가용 시간표 기반 캘린더 UI, 이미 예약된 슬롯 비활성화 Must
문진표 작성 예약 신청 필수 단계 (건너뛰기 불가)
  • 주 호소 내용 — 자유 서술 (최대 1,000자)
  • 심각도 자가평가 — 1~5 점수 선택
  • 지속 기간 — 선택형 (1주 이내 / 1개월 이내 / 3개월 이상 등)
  • 이전 상담 경험 여부 — 예 / 아니오
  • 상담 목표 — 자유 서술 (최대 500자)
 Must
결제 처리 토스페이먼츠 또는 포트원 연동 (국내 PG)
  • 결제 수단: 신용카드, 계좌이체
  • 결제 실패 시 예약 취소 처리
  • 결제 완료 확인 메일 자동 발송
 Must
채팅방 자동 생성 결제 완료 즉시 해당 내담자-상담사 전용 채팅방 생성 및 활성화 Must
4
채팅
기능 항목상세 설명우선순위
실시간 채팅 WebSocket(STOMP) 기반, 결제 완료 후 채팅방 활성화 Must
텍스트 메시지 메시지 전송·수신, 읽음 처리, 타임스탬프 표시 Must
링크 공유 URL 자동 감지 및 링크 미리보기 (Zoom 링크 등) Must
파일 전송 이미지·문서 파일 업로드 및 다운로드 (최대 10MB) Should
채팅 이력 보존 세션 종료 후에도 채팅 이력 영구 보존 및 조회 가능 Must
5
세션 관리
기능 항목상세 설명우선순위
예약 내역 조회 상태별 탭 구분 — 예정 / 완료 / 취소 Must
취소 요청 내담자가 직접 취소 요청 가능
  • 세션 24시간 전 취소: 전액 환불
  • 세션 24시간 이내 취소: 50% 환불
  • 세션 시작 후: 취소 불가 → 노쇼 신고 경로로 전환
  • 취소 정책 안내 팝업 표시 필수
 Should
상담사 귀책 취소·노쇼 상담사가 시점 무관 취소하거나 나타나지 않은 경우 (12번 마스터 플랜 ADR-7 — 오너 결정 2026-07-07)
  • 내담자 전액 환불 (시점 무관)
  • 당일취소·노쇼는 상담사 페널티 누적 — 금전 페널티 없음, 30일 내 3회 누적 시 검색 노출만 자동 제외 + 관리자 알림
  • 노쇼 판정은 자동화하지 않음 — 내담자가 노쇼 신고 → 관리자가 분쟁 해결(전액환불/반려) 결정
 Must
상담 완료 처리 세션 시간 경과 후 '완료' 상태 자동 전환, 리뷰 작성 유도 Must
6
리뷰
기능 항목상세 설명우선순위
리뷰 작성 세션 완료 후 작성 가능 — 별점 1~5 + 텍스트 (최대 500자) Must
익명 옵션 리뷰 작성 시 익명 공개 여부 선택 가능 (필수 지원) Must
리뷰 수정 / 삭제 작성자 본인만 수정·삭제 가능 (작성 후 7일 이내) Should
7
마이페이지
기능 항목상세 설명우선순위
내 정보 수정 이름, 생년월일, 비밀번호 변경, 프로필 사진 업로드 Must
예약 이력 전체 예약 내역 목록 (예정·완료·취소 포함) Must
채팅 목록 활성화된 채팅방 목록, 최근 메시지 미리보기, 읽지 않은 메시지 수 표시 Must
즐겨찾기 상담사 관심 상담사 북마크 및 목록 관리 Could

상담사 (Counselor) 기능

임상·상담심리 석사 이상 자격 보유자 — 관리자 심사 후 활동 가능

1
가입 & 자격 인증
기능 항목상세 설명우선순위
이메일 가입 이메일 + 비밀번호, 이메일 인증 절차 동일 적용 Must
자격 서류 업로드 다음 서류 중 해당 항목 업로드 (이미지·PDF, 각 최대 10MB)
  • 재학증명서 (석사 과정 재학 중인 경우)
  • 학위증명서 (졸업자)
  • 자격증 사본 (임상심리사 1·2급, 상담심리사 1·2급 등)
  • 신분증 사본 (주민등록증 또는 운전면허증)
 Must
심사 대기 상태 표시 업로드 완료 후 심사 대기 화면 노출, 예상 소요 시간(최대 72시간) 안내 Must
승인 결과 알림 승인 또는 거절 결과를 이메일로 자동 발송 (거절 시 사유 포함) Must
재심사 요청 거절 후 서류 재업로드 및 재신청 가능 Should
2
프로필 관리
기능 항목상세 설명우선순위
프로필 사진 업로드 JPG·PNG 형식, 최대 5MB, 자동 리사이징 처리 Must
자기소개 작성 마크다운 미지원 순수 텍스트, 최대 2,000자 Must
전문 분야 선택 v1 기준: 우울 / 번아웃 (복수 선택 가능) Must
상담 방식 선택 채팅 / 화상링크 (복수 선택 가능) Must
상담 접근법 선택 CBT, ACT, 인간중심치료, DBT, 정신분석 등 (복수 선택) Should
가격 설정 세션당 KRW 입력, 최소 30,000원 권고 (강제 하한선 적용 여부는 정책 결정 필요) Must
자격 뱃지 자동 표시 관리자 승인 완료 시 프로필에 '인증 상담사' 뱃지 자동 부착 Must
3
가용 시간 관리
기능 항목상세 설명우선순위
주간 반복 스케줄 설정 요일별 시작·종료 시간 설정, 1시간 단위 슬롯 자동 생성 Must
특정 날짜 추가 주간 스케줄 외 특정 날짜·시간 수동 추가 Should
특정 날짜 차단 휴가·부재 등으로 특정 날짜 전체 또는 시간대 비활성화 Should
4
예약 관리
기능 항목상세 설명우선순위
신규 예약 알림 새 예약 결제 완료 시 이메일 알림 발송 Must
예약 확인 신규 예약 확인 처리 (확인 완료 상태 전환) Must
예약 거절 사유 입력 필수 (최대 200자), 24시간 이내 응답 권고
  • 거절 시 내담자에게 이메일 발송 및 자동 전액 환불 처리
 Must
예약 현황 달력 뷰 월간 달력에 예약 상태(확인 대기 / 확인 완료 / 완료 / 취소) 색상 구분 표시 Should
5
문진표 열람
기능 항목상세 설명우선순위
결제 완료 세션만 접근 가능 해당 내담자가 결제를 완료한 세션의 문진표만 열람 허용 Must
읽기 전용 문진표 내용 수정·삭제 불가, 출력 기능은 별도 정책 결정 필요 Must
접근 제어 본인 세션 외 타 상담사 세션의 문진표 접근 차단 (서버 사이드 권한 검증) Must
6
채팅
기능 항목상세 설명우선순위
내담자와 실시간 채팅 WebSocket(STOMP), 전송·수신 UI 내담자와 동일한 채팅방 공유 Must
링크 공유 Zoom 또는 기타 화상회의 링크 텍스트 전송 (자동 하이퍼링크 처리) Must
7
수익 & 정산
기능 항목상세 설명우선순위
세션별 수익 내역 총 결제금액 / 플랫폼 수수료 / 실수령액 표시 Should
월별 정산 내역 월 단위 집계 — 세션 수, 총액, 수수료, 정산 예정액 Should
정산 계좌 등록 은행명, 계좌번호, 예금주명 입력 및 인증 (계좌 인증 로직 별도 정의 필요) Should

관리자 (Admin) 기능

플랫폼 운영·심사·분쟁 처리 담당 내부 관리자

1
상담사 자격 심사
기능 항목상세 설명우선순위
심사 대기 목록 접수 순 정렬, 신청일·이름·제출 서류 수 표시 Must
서류 뷰어 이미지·PDF 인라인 표시 (다운로드 없이 브라우저 내 열람)
  • 재학증명서 / 학위증명서 / 자격증 / 신분증 탭 구분
 Must
승인 처리 승인 클릭 → 상담사 계정 활성화, 자격 뱃지 자동 부착, 결과 이메일 발송 Must
거절 처리 사유 입력 필수 → 상담사에게 거절 사유 포함 이메일 자동 발송 Must
2
회원 관리
기능 항목상세 설명우선순위
전체 회원 목록 조회 내담자 / 상담사 탭 구분, 검색(이름·이메일), 가입일 필터 Must
계정 정지 사유 입력 후 계정 접근 차단, 정지 이유 이메일 발송 Must
탈퇴 처리 개인정보 삭제 처리 (서비스 이용 데이터는 법적 보존 기간 내 보관) Must
3
세션 & 결제 관리
기능 항목상세 설명우선순위
전체 세션 내역 조회 상태(예정 / 완료 / 취소), 날짜 범위, 상담사·내담자명 기준 필터 Must
결제 내역 조회 결제 금액, PG 승인 번호, 결제 수단, 상태(정상 / 환불) 표시 Must
4
분쟁 처리
기능 항목상세 설명우선순위
환불 강제 실행 관리자가 직접 특정 세션에 대해 전액·부분 환불 처리 권한 보유 Should
분쟁 이력 기록 분쟁 접수·처리 내용을 이력으로 저장, 관리자 전용 열람 Should
5
통계 대시보드
기능 항목상세 설명우선순위
총 가입자 수 상담사 / 내담자 구분 표시, 누적 및 신규(이번 달) 수치 Should
월별 세션 수 월 단위 완료 세션 수 차트 (막대 그래프) Should
월별 거래액 / 플랫폼 수익 총 결제액 및 수수료 수익 추이 (월별 라인 차트) Should
상담사 승인 현황 심사 대기 / 승인 완료 / 거절 건수 현황 카드 Should

MoSCoW 우선순위

MVP 범위 결정 및 릴리즈 단계 계획 기준

Must Have
MVP 출시 필수 — 없으면 서비스 불가
  • 상담사 등록 및 자격 인증 (관리자 심사)
  • 내담자 회원가입 및 이메일 인증
  • 상담사 목록·프로필 탐색·필터
  • 예약 및 문진표 작성
  • 결제 (토스페이먼츠 / 포트원)
  • 결제 완료 후 채팅방 자동 생성
  • 실시간 채팅 (WebSocket / STOMP)
  • 기본 리뷰 작성 (별점 + 텍스트 + 익명)
  • 관리자 자격 심사 화면 (승인 / 거절)
  • 채팅 이력 보존
Should Have
v1.0 릴리즈 포함 — 중요하나 MVP와 분리 가능
  • 취소 / 환불 자동화 (24시간 정책)
  • 상담사 정산 내역 및 계좌 등록
  • 이메일 알림 (신규 예약, 심사 결과)
  • 관리자 통계 대시보드
  • 예약 현황 달력 뷰 (상담사)
  • 파일 전송 (채팅)
  • 상담사 재심사 요청
  • 리뷰 수정 / 삭제
  • 분쟁 처리 (환불 강제 실행)
Could Have
v2.0 이후 — 리소스 여유 시 포함
  • 푸시 알림 (브라우저 / 앱)
  • 즐겨찾기 상담사
  • AI 기반 상담사 추천
  • 그룹 상담 기능
  • 소셜 로그인 (구글 OAuth)
  • 상담 접근법 세부 선택
Won't Have
MVP 제외 — 명시적 비포함 결정
  • 모바일 앱 (iOS / Android 네이티브)
  • 자체 영상통화 구현
  • 자격증 OCR 자동 검증
  • 카카오 / Zoom API 직접 연동
  • 다국어 (영문) 지원

기술 제약 및 정책 사항

기술 스택

  • Backend: Java Spring Boot 3.x
  • DB: MySQL
  • Frontend: React 18 + TypeScript
  • 실시간: Spring WebSocket (STOMP)

결제

  • 국내 PG: 토스페이먼츠 또는 포트원
  • 외부 결제 API 미사용 (Stripe 등 제외)
  • 환불: PG 취소 API 자동 연동 필요

외부 API 제한

Zoom API, 카카오 API 등
외부 서비스 직접 연동 없음
Zoom 링크는 수동 입력·전송

상담사 자격 기준

임상·상담심리 석사 이상
자격증·학위 서류 업로드 후
관리자 수동 심사 (최대 72h)

데이터 보안

  • 문진표: 결제 완료 상담사만 열람
  • 채팅: 당사자 외 접근 금지
  • 서류: 관리자 전용 접근

취소 / 환불 정책

  • 내담자 24시간 전 취소: 전액 환불
  • 내담자 24시간 이내 취소: 50% 환불
  • 내담자 시작 후 취소: 불가 (노쇼 신고 경로)
  • 상담사 거절·취소·노쇼: 시점 무관 전액 환불 + 페널티 누적 (ADR-7, 12번 §3)
